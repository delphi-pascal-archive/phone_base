Для уменьшения размеров, программа распространяется без индексных файлов.
Они будут созданы в момент первого запуска.
При запуске программы ищется индексный файл Data.MDX и если не находится, то в окремому
потоке DataThread создается снова тимчасова база, которая потом переименовывается в Data.DBF
и Data.MDX.
Для открытия базы без индексного файла, процедура RemaskMDX заменяет відповідний бит в файле Data.DBF.
